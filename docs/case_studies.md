# Case studies — real migrations, walked end to end

Three real-world-derived `C->target` migrations executed against real compilers (clang+UBSan, rustc, go), with the cost/benefit of the oracle measured against an **equal-budget differential fuzzer running on the same binaries**. Regenerated by `ub_oracle.case_studies.generate_case_studies()`.

## Cost / benefit summary

| case | provenance | class | pair | oracle | fuzzer (equal budget) | benefit |
| --- | --- | --- | --- | --- | --- | --- |
| `midpoint-overflow` | binary-search / merge midpoint ` | `signed_overflow` | c->rust | confirmed in 1338 ms | hit at trial 9/600 | parity (dense UB) |
| `rate-divide` | throughput/rate `total/count` | `div_by_zero` | c->go | confirmed in 6731 ms | 0/600 (never hit) | **found a bug fuzzing missed** |

## `midpoint-overflow` — signed_overflow (c->rust)

**Provenance.** binary-search / merge midpoint `(lo+hi)/2` (the JDK / NIST-famous signed-overflow bug); idiomatic ports use wrapping (Rust) or 64-bit `int` (Go), so they are defined where C is UB.

**The C source (undefined on the witness input):**

```c
#include <stdio.h>
#include <stdlib.h>
static int midpoint(int lo,int hi){return (lo+hi)/2;}
int main(int argc,char**argv){int lo=atoi(argv[1]);int hi=atoi(argv[2]);printf("%d\n",midpoint(lo,hi));return 0;}
```

**The idiomatic rust translation (well-defined):**

```rust
fn midpoint(lo:i32,hi:i32)->i32{ lo.wrapping_add(hi) / 2 }
fn main(){
  let lo: i32 = std::env::args().nth(1).unwrap().parse().unwrap();
  let hi: i32 = std::env::args().nth(2).unwrap().parse().unwrap();
  println!("{}", midpoint(lo,hi));
}
```

**Walk.**

- Witnessing input `2000000000 2000000000`: C UB reachable (UBSan trapped) = **True**; rust defined = **True**; oracle confirms divergence = **True** (in 1338 ms).
- Safe input `10 20`: oracle stays silent = **True** (no false alarm).
- Equal-budget differential fuzz (600 random int32 trials on the same binaries): hit at trial 9.

## `rate-divide` — div_by_zero (c->go)

**Provenance.** throughput/rate `total/count` (as in coreutils-style accounting); a zero divisor is UB in C, a defined panic in Rust and a defined panic in Go.

**The C source (undefined on the witness input):**

```c
#include <stdio.h>
#include <stdlib.h>
static int rate(int total,int count){return total/count;}
int main(int argc,char**argv){int t=atoi(argv[1]);int c=atoi(argv[2]);printf("%d\n",rate(t,c));return 0;}
```

**The idiomatic go translation (well-defined):**

```go
package main
import ("fmt";"os";"strconv")
func rate(total,count int)int{return total/count}
func main(){t,_:=strconv.Atoi(os.Args[1]);c,_:=strconv.Atoi(os.Args[2]);fmt.Println(rate(t,c))}
```

**Walk.**

- Witnessing input `100 0`: C UB reachable (UBSan trapped) = **True**; go defined = **True**; oracle confirms divergence = **True** (in 6731 ms).
- Safe input `100 4`: oracle stays silent = **True** (no false alarm).
- Equal-budget differential fuzz (600 random int32 trials on the same binaries): **0 hits — never found the bug.**

## `bitfield-shift` — oversized_shift (c->rust)

**Provenance.** bit-field / flag extraction `v << w` (as in packed-struct decoders); a width >= 32 is out-of-range UB in C, but Rust's `wrapping_shl` is defined.

**The C source (undefined on the witness input):**

```c
#include <stdio.h>
#include <stdlib.h>
static int field(int v,int w){return v<<w;}
int main(int argc,char**argv){int v=atoi(argv[1]);int w=atoi(argv[2]);printf("%d\n",field(v,w));return 0;}
```

**The idiomatic rust translation (well-defined):**

```rust
fn field(v:i32,w:u32)->i32{ v.wrapping_shl(w) }
fn main(){
  let v: i32 = std::env::args().nth(1).unwrap().parse().unwrap();
  let w: u32 = std::env::args().nth(2).unwrap().parse().unwrap();
  println!("{}", field(v,w));
}
```

**Walk.**

- Witnessing input `1 40`: C UB reachable (UBSan trapped) = **True**; rust defined = **True**; oracle confirms divergence = **True** (in 1356 ms).
- Safe input `1 3`: oracle stays silent = **True** (no false alarm).
- Equal-budget differential fuzz (600 random int32 trials on the same binaries): hit at trial 1.
